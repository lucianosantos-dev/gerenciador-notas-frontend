# 📑 Gerenciador de Notas Fiscais — Frontend

![Angular](https://img.shields.io/badge/angular-%23DD0031.svg?style=for-the-badge&logo=angular&logoColor=white)
![TypeScript](https://img.shields.io/badge/typescript-%23007ACC.svg?style=for-the-badge&logo=typescript&logoColor=white)

> 🚧 **Projeto em desenvolvimento** — interface ainda incompleta, em evolução.

Interface web (Angular) para consumo da [API de Gerenciamento de Notas Fiscais](https://github.com/lucianosantos-dev/Gerenciador-Nota-fiscais), permitindo visualizar, organizar e controlar o vencimento de notas fiscais de forma prática.

## ✅ O que já funciona

- Estrutura inicial do projeto Angular configurada

## 🔧 Em progresso

- Integração completa com a API backend
- Telas de listagem e cadastro de notas
- Estilização final da interface

## 🛠️ Tecnologias

- Angular 21
- TypeScript

## ⚙️ Como rodar localmente

```bash
git clone https://github.com/lucianosantos-dev/gerenciador-notas-frontend.git
cd gerenciador-notas-frontend
npm install
ng serve
```

Acesse em `http://localhost:4200/`
